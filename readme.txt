== Description ==

A plugin for WordPress to display html5 videos.

= Feedback =


== Installation ==

1. Search for `html5 video` in the `Plugins` menu to install.
1. Or upload folder html5video to the `/wp-content/plugins/` directory.
1. Activate html5 video through the 'Plugins' menu in WordPress.
1. Place the code such as `[video src="http://www.sample.com/xxx.mp4"]` in your post, or just use the `media Insert` in TinyMCE, and then publish it.
